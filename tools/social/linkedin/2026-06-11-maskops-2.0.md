## Post — English (version)

This week I shipped MaskOps 2.0. Before I told anyone, I looked at my own numbers. Two were lying to me — in opposite directions.

MaskOps is a Rust plugin for Polars that masks PII inside the dataframe: RUT, CPF, CURP, cards, IBAN, twenty more families. No network call, ever. Check-digit validation, so a random nine-digit number isn't mistaken for an ID.

The first number lied against me. My own benchmark said it was slower than plain Python. The bug wasn't the code — it was the measurement: it ran all 35 families against a Python baseline running one. Measured fairly: 11–163× on clean data, 1.2–3.2× on mixed.

The second lied for me. PyPI downloads jumped 100× on release day. Easy to write "explosive growth." Also false: it's CI, mirrors, and bots, not people. I said so in the article.

A metric isn't a verdict. It's a measurement, and it can be miscalibrated in your favor or against it. Worth knowing which.
